# Whistle

**AI-powered real-time trading intelligence for FIFA World Cup 2026**

Built for the [TxODDS World Cup Hackathon](https://txline.txodds.com/documentation/legal/hackathon-terms) — Track 2: Trading Tools & Agents.

## What it does

Whistle monitors dual TxODDS SSE streams (live odds + match events) simultaneously, correlates them through a divergence detection engine, and delivers AI-narrated alerts via Telegram when it spots a mismatch between what's happening on the pitch and what the market is pricing.

### Divergence patterns detected

| Pattern | What it means | Severity |
|---------|--------------|----------|
| **Silent Odds Shift** | Multiple bookmakers move odds sharply with no visible match event | Critical/High |
| **Delayed Market Reaction** | Goal/red card/penalty happened but odds haven't adjusted in 10s+ | High |
| **Momentum Mispricing** | 3+ consecutive danger possessions but odds haven't shortened | Medium |
| **Bookmaker Disagreement** | >15% spread across bookmakers on the same market | Medium |
| **Goal Imminent** | TxODDS flags imminent goal but over/BTTS market hasn't tightened | Medium |
| **Odds Collapse** | Odds crash from >1.50 to <1.20 — outcome near-certain | Critical |

## Tech stack

- **Runtime**: Node.js + TypeScript
- **TxODDS**: Dual SSE streams (odds + scores) via TxLINE API
- **Solana**: On-chain subscription via Anchor (devnet, Token-2022)
- **AI**: LLaMA 3.3 70B via OpenRouter (free tier)
- **Bot**: Grammy (Telegram bot framework)
- **Deployment**: Railway
- **Dashboard**: Next.js (landing page + live alert feed)

## Setup

### Prerequisites

- Node.js 18+
- A Telegram bot token (from [@BotFather](https://t.me/BotFather))
- An OpenRouter API key (free at [openrouter.ai](https://openrouter.ai))
- Devnet SOL (~0.01 for transaction fees)

### Quick start

```bash
# Install dependencies
npm install

# Run the setup script (handles Solana wallet, on-chain subscription, API token)
npm run setup

# Build and run
npm run build
npm start
```

The setup script will:
1. Generate a Solana keypair (or load existing from `.env`)
2. Get a guest JWT from TxODDS
3. Subscribe on-chain (free World Cup tier via Token-2022)
4. Activate your API token
5. Save all credentials to `.env`

### Environment variables

```
TELEGRAM_BOT_TOKEN=       # From @BotFather
OPENROUTER_API_KEY=       # From openrouter.ai
TXODDS_JWT=               # Auto-set by setup script
TXODDS_API_TOKEN=         # Auto-set by setup script
SOLANA_PRIVATE_KEY=       # Auto-generated by setup script
SOLANA_RPC_URL=https://api.devnet.solana.com
```

## How the divergence engine works

The core insight: **odds and events should move together**. When they diverge, there's either an information asymmetry or a market inefficiency.

1. **OddsTracker** processes each bookmaker's price update, calculates velocity (% change over 60s window), cross-bookmaker spread, and detects collapses
2. **EventTracker** processes score events — goals, cards, VAR reviews, danger possessions, phase changes
3. **DivergenceDetector** cross-correlates both signal streams within a 2-minute window to detect the 6 patterns above
4. Alerts have a 3-minute cooldown per fixture per type to avoid spam
